# Retinal Disease Detector

A deep learning web app for detecting diabetic retinopathy from fundus images.

## Results
- Quadratic Weighted Kappa: 0.9053
- Accuracy: 83.6%

Upload any retinal fundus image to get an instant diabetic retinopathy severity grade (0-4).
